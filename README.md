# Numerical Linear Algebra Toolkit

A Python implementation of fundamental numerical linear algebra algorithms from scratch, built for educational and scientific computing purposes.

This project demonstrates core matrix algorithms used in computational mathematics, physics, machine learning, and quantum mechanics.

---

## Features

### Implemented Algorithms

• Gaussian Elimination (solving Ax = b)  
• LU Decomposition (A = L @ U)  
• QR Decomposition using Gram–Schmidt process (A = Q @ R)  
• Power Method for dominant eigenvalue and eigenvector computation
